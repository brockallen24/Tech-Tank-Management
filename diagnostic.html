Required Configuration

What You Need in Heroku Config Vars:

1. AIRTABLE_API_KEY - Your Airtable Personal Access Token

  • Get it from: https://airtable.com/create/tokens
  • Should start with pat
  • Needs scopes: data.records:read, data.records:write, schema.bases:read

2. AIRTABLE_INVENTORY_BASE_ID - Your Inventory Base ID

  • Found in your Airtable URL
  • Should start with app
  • Example: appAbC123DefGhi45
